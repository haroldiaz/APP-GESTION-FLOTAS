# NEOFLOTA \
## APP-GESTION-FLOTAS <br/>


## Caracteristicas <br/>

Esta aplicación permite gestionar de forma sencilla toda la información relacionada con el transporte. 
 - Registro de Usuarios
 - Agregar conductores
 - Asociar vehículos 
 - Administrar los diferentes servicios realizados.  <br/>

Está diseñada para empresas o plataformas que necesiten llevar un control completo y organizado de su flota, conductores y operaciones diarias. <br/>


## Tecnologias Utilizadas <br/>
kotlin\
JeckPack Compose\
